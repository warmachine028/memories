## Tech Stack Used

-   Material UI: Styling & Icons
-   Postgres: Database
-   Elysia: Backend Framework
-   Bun: JavaScript Runtime
-   Vercel: Frontend Hosting
-   Render: Backend Hosting
-   Clerk: Authentication
